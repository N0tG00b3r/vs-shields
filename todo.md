## 🔊 Задачи для дизайнера (Звук)

> Файлы кладутся в `common/src/main/resources/assets/vs_shields/sounds/`, формат OGG Vorbis, моно, 44100 Hz

### Варианты по тирам (приоритет)
- [ ] **`shield_hum_iron.ogg`** — тихий низкочастотный гул, как трансформатор
- [ ] **`shield_hum_diamond.ogg`** — чистый высокий резонанс, кристаллический тон
- [ ] **`shield_hum_netherite.ogg`** — тяжёлый глубокий бас, ощущение массы
- [ ] *(код: выбор hum по тиру в `ShieldAmbientSoundHandler`)*

### Новые события
- [ ] **`shield_stress.ogg`** — непрерывный crackling/электрический треск для щита под огнём (HP < 25%); loopable
- [ ] **`shield_reboot.ogg`** — звук восстановления после depletion-кулдауна; нарастающий power-up
- [ ] **`shield_low_energy.ogg`** — короткий предупреждающий бип когда FE < 20% (раз в ~5 сек)
- [ ] **`jammer_activate.ogg`** — начало работы глушителя: нарастающий interference whine + static burst
- [ ] **`jammer_fire.ogg`** — момент «выстрела» EW: sonic boom + electronic screech
- [ ] **`gravity_field_hum.ogg`** — фоновый гул гравитационного поля; глубокий space-bass, loopable
- [ ] **`battery_charge.ogg`** — тихий тон при зарядке батареи (раз в несколько секунд при активной зарядке)
- [ ] **`shield_overcharge.ogg`** — звук активации перегрузки (будущая фича); мощный charge-up

### Требования к звукам
- Моно-канал (позиционирование в мире через Minecraft)
- Нормализация: peak не выше −3 dBFS
- Длина: hit/collapse/activation — 0.5–2 сек; hum/stress — бесшовный луп 3–8 сек
- Стиль: sci-fi / industrial, без музыкальных мотивов
