# VS Energy Shields — Идеи по расширению

## Новые блоки и компоненты

### Частотный тюнер щита (Shield Frequency Tuner)
- Позволяет несколько щитов на одном корабле (разные частоты не конфликтуют)
- Каждая частота защищает свою зону корабля
- Крафт: алмазы + эндер-жемчуг + генератор

### Ретранслятор щита (Shield Relay)
- Расширяет дальность щита без увеличения HP
- Цепочка ретрансляторов для покрытия на большие расстояния
- Полезен для крупных кораблей, где радиуса не хватает

### ~~Батарея щита (Shield Battery)~~ ✅ РЕАЛИЗОВАНО
- 3x3x3 мультиблок: Controller + Cell + Input
- Пассивная регенерация 20% + экстренный сброс при <25% HP
- FE + Create SU через Input блоки

### ~~Генератор маскировочного поля (Cloaking Field Generator)~~ ✅ РЕАЛИЗОВАНО
- Делает корабль невидимым, высокий расход энергии
- CloakShimmerRenderer для визуального эффекта

## Игровые механики

### Модуляция / Адаптивные щиты
- Щит "учится": повторный урон того же типа наносит всё меньше урона
- Сбрасывается когда HP падает до 0
- Визуал: цвет щита смещается в зависимости от активной модуляции

### Режим перегрузки (Overcharge)
- Переключатель: жертвуешь HP щита ради временного наступательного баффа
- Пока перегружен, существа внутри щита получают эффект Силы
- Щит тратится в 2 раза быстрее, не перезаряжается

### Гармоники щита
- Несколько конденсаторов одного тира создают "гармонический бонус" (экспоненциальное масштабирование)
- 1 конд. = +50 HP, 2 = +120, 3 = +200, 4 = +300
- Стимулирует вложения в инфраструктуру корабля

### Пробивание щита (Shield Piercing)
- Некоторые типы снарядов частично обходят щиты (напр. трезубец пробивает 50%)
- Поощряет разнообразие вооружения в PvP
- Настраиваемый процент пробития на каждый тип снаряда

### Направленные щиты
- Эмиттеры концентрируют прочность щита в направлении, куда смотрят
- Передняя часть корабля получает 2x защиту, задняя 0.5x
- Визуально: более яркая секция на щите

## Визуал и звук

### Эффекты попадания (Impact VFX)
- Всплеск частиц при попадании (рябь в точке удара)
- Цвет соответствует цвету здоровья щита
- Больший всплеск для тяжёлых ударов

### Анимация разрушения щита
- Когда HP падает до 0: эффект разбитого стекла + ударная волна частиц
- Звук: электрический треск + звук разбитого стекла
- 3-секундный визуальный "обломок щита"

### Фоновый гул щита
- Зацикленный звук рядом с активным щитом
- Высота тона растёт при падении HP
- Интенсивность увеличивается у границы щита

### Мерцание при низкой энергии
- Когда FE < 20%, щит визуально глючит/мерцает
- Увеличивается прозрачность, появляются разрывы в рендере
- Предупреждает игрока о необходимости дозаправки

## Интеграция / Совместимость

### Create: щиты на основе вращения
- Альтернатива FE: щит питается напрямую от вращательного стресса
- Отдельный блок "Кинетическое ядро щита" подключается к валам Create
- Требуемые SU масштабируются с тиром щита

### Computercraft / CC:Tweaked
- Peripheral API: `getShieldHP()`, `getMaxHP()`, `getEnergy()`, `setActive(bool)`
- Автоматическое управление щитом через Lua-скрипты
- События: `shield_hit`, `shield_break`, `shield_recharge_complete`

### Eureka! (штурвал корабля)
- Отображение состояния щита в GUI штурвала
- Автоактивация щита когда штурвал занят
- Приоритетная маршрутизация энергии через штурвал

### WITS (интеграция оружия)
- Совместимость с другими оружейными аддонами VS2
- Стандартизированное API урона для модовых снарядов
- Конфиг-файл для настройки урона по типам entity

## Конфигурация и кастомизация

### Конфиг-файл (vs_shields-common.toml)
- Значения HP/радиуса/перезарядки/кулдауна/энергии для каждого тира
- Фактор урона от взрывов (сейчас 5.5)
- Переопределение урона по типам entity
- Макс. число конденсаторов/эмиттеров на корабль
- Включение/отключение отдельных фич

### Скины / цвета щита
- Окраска щита красителями (применить краситель к генератору)
- Поддержка текстурпаков для рисунка поверхности щита
- Форма щита (сфера, куб, цилиндр) через конфиг

### Масштабирование сложности
- Лёгкая: щиты перезаряжаются быстрее, больше HP
- Сложная: щиты потребляют больше энергии, медленнее перезарядка
- Серверный конфиг для PvP-балансировки

## Продвинутые фичи (долгосрочные)

### Сеть щитов (Shield Network)
- Связь щитов нескольких кораблей через блоки "Shield Link"
- Урон по одному щиту распределяется по всей сети
- У сети общий пул HP, разделённый между всеми кораблями

### Отражение снарядов
- При >75% HP щит с шансом отражает снаряды
- Отражённые снаряды летят в случайном направлении
- Визуал: искра рикошета

### Уязвимость к ЭМИ (EMP)
- Новый тип оружия: ЭМИ бомба/торпеда
- Обходит HP щита, напрямую истощает энергию
- Контрмера: блок "Защита от ЭМИ"

### Скремблер щита (наступательный)
- Устанавливаемое устройство, ослабляющее вражеские щиты поблизости
- Снижает скорость перезарядки и макс. HP в радиусе действия
- Контрится частотным тюнером

### Варп-щит
- Телепортирует снаряды, попавшие в щит, в случайную позицию поблизости
- Очень высокий расход энергии
- Редкий рецепт крафта (эндгейм)
